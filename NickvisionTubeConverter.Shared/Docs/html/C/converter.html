Converter

This page explains all options for configuring the Converter in Preferences in Tube Converter.

Disallow Conversions

If enabled, Tube Converter will download the appropriate video/audio format for the selected quality without converting to other formats. This means that you will be unable to select a file format when downloading a media. But, you will save time in downloading as the Converter will not be run.

Default: Off

Embed Metadata

If enabled, Tube Converter will tag a media with its relevant information (title, genre, etc...) and thumbnail if available. Tube Converter will NOT embed the Comment, Description, Synopsis, and Purl fields.

Default: On

Crop Audio Thumbnails

Only available if Embed Metadata is enabled. If enabled, Tube Converter will automatically turn on the advanced option "Crop Thumbnail" for audio downloads.

Default: Off

Embed Chapters

If enabled, Tube Converter will tag a media with chapter information if available

Default: Off
